Weather App (Python)

A simple CLI weather application using OpenWeather API.

Features

- Get real-time weather data
- Shows temperature, condition, and humidity
- Works for any city

Tech

- Python
- Requests library
- OpenWeather API

How to Run

1. Install requests:
   pip install requests

2. Add your API key in the code:

- Sign Into OpenWeather
- Navigate to API Keys
- Select "My API keys"
- Copy the key

3. Run:
   python weather.py
